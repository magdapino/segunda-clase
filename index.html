3 TIPOS DE GRÁFICAS PARA REPRESENTAR INFORMACIÓN

Magdalena Pino & Anotnia Moreno

Cuando queremos representar cualquier tipo de información vamos a encontrar infinitas opciones, en este caso veremos tres tipos de gráficas: figurativos, no figurativos y mixtos.

GRÁFICA FIGURATIVA

Estas cuatro imágenes corresponden a gráficas figurativas. Estas corresponden a información que se puede observar y comprender.

GRÁFICA MIXTA

Como dice el nombre, esta gráfica combina gráficas figurativas y no figurativas.

GRÁFICA NO FIGURATIVA

Este tipo de gráfica corresponde a los gráficos utilizados para representar información que no se puede observar, como lo pueden ser números y estadísticas.

La gráfica de arriba se despliega en la página como si se tratara de una imagen rasterizada, con el elemento <img>. Otros elementos de HTML podrían usarse para un mejor despliegue del SVG. Reviseramos tales elementos algunas clases más adelante, pero ya les conviene revisar Using SVG | CSS Trick.
